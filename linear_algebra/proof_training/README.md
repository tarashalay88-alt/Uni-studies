# Linear Algebra Proof Training

Goal:

> See an unfamiliar statement, classify it, choose the right proof weapon, and execute cleanly.

## Daily cycle

1. Micro-proofs from memory.
2. One arbitrary statement.
3. One counterexample.
4. One correction-note entry.

## Core proof weapons

- double inclusion
- existence + uniqueness
- kernel trick
- image trick
- dimension argument
- basis extension
- direct-sum decomposition
- adjoint/orthogonal complement identities
