# Uni Studies

Personal university preparation repository for MFDS.

Main principle:

> Do not just collect notes. Produce understanding, proofs, solved problems, code, and exam-ready summaries.

## Structure

```text
linear_algebra/
  theory/
  problem_sheets/
  coding/
  proof_training/
  exam_prep/

analysis/
  theory/
  problem_sheets/
  coding/
  exam_prep/

algorithms_and_data_structures/
  theory/
  problem_sheets/
  coding/
  exam_prep/
```

## Rules

1. Do not upload copyrighted lecture PDFs or full copied notes.
2. Use lecture notes as references, but write summaries and proofs in your own words.
3. Every solved problem should have:
   - first attempt,
   - corrected version,
   - mistake note,
   - reusable idea.
4. For Linear Algebra, proof training is mandatory.
5. For Algorithms and Data Structures, every concept should eventually have code.
